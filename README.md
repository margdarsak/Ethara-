EtharaTask - Team Task Manager
==============================

A full-stack project management application built with React, Node.js, and Prisma.

Features:
---------
- User Authentication (Admin & Member roles)
- Project Management
- Task Tracking (Todo, In Progress, Done)
- Responsive Dashboard with stats
- Animated Marquee UI components
- Role-based Access Control

Tech Stack:
-----------
- Frontend: React, Vite, Lucide Icons, Vanilla CSS
- Backend: Node.js, Express, Prisma ORM
- Database: PostgreSQL (Supabase)

Setup Instructions:
-------------------
1. Backend:
   - Go to 'server' folder
   - Run 'npm install'
   - Create '.env' with your DATABASE_URL
   - Run 'npx prisma db push'
   - Run 'npm run dev'

2. Frontend:
   - Go to 'client' folder
   - Run 'npm install'
   - Run 'npm run dev'

3. Access the app at https://etharataskmanager.up.railway.app/
